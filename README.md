# 🚀 Campus Connect

Campus Connect is a full-stack student collaboration platform designed to help students communicate, share resources, and collaborate efficiently in one place.

## 🌟 Features

- 💬 Real-time student chats
- 👥 Study group creation and management
- 📂 File sharing (notes, PDFs, resources)
- 📢 Announcements system
- 🧑‍💼 Admin dashboard for monitoring and control
- 📱 Responsive design (mobile + desktop)

## 🛠️ Tech Stack

- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- Authentication: JWT-based auth

## 🎯 Purpose

This project was built to simplify student collaboration by combining multiple tools (chat, file sharing, group discussions) into a single platform.

## 🚀 Getting Started

### Clone the repo
git clone https://github.com/jeedijoshua-art/campus-connect.git

### Install dependencies
cd campus-connect
npm install

### Run the app
npm run dev
